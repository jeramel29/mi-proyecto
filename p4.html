Que es BOOTSTRAP?

Bootstrap es un potente marco frontal para un desarrollo web más rápido y fácil. Incluye plantillas de diseño basadas en HTML y CSS para crear componentes comunes de la interfaz de usuario como formularios, botones, navegaciones, menús desplegables, alertas, modales, pestañas, acordeones, carruseles, información sobre herramientas, etc. Bootstrap le brinda la capacidad de crear diseños web flexibles y receptivos con mucho menos esfuerzo.
